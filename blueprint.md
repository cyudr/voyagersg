# Travel Planner App PRD

## Overview
A web-based travel planning tool that transforms a user's interests, budget, and dietary preferences into a personalized, multi-day itinerary.

## User Workflow
1. **Landing / Start**: Begin planning (e.g., "Singapore visit").
2. **Preference Discovery**:
    - **Interests**: Culture, Nature, Modern.
    - **Budget**: Daily limit setting.
    - **Dietary**: Preference selection.
3. **Selection Phase**:
    - **Places**: Select specific spots by interest.
    - **Food**: Choose dining experiences.
    - **Special Extras**: Choose unique local experiences.
4. **Analysis & Length**: System suggests trip length; user confirms days.
5. **Generation**: Finalized personalized itinerary.
6. **Review & Finalize**: Adjust preferences if needed or finalize the trip.

## Visual Direction
Inspired by Wanderlog: Clean, card-based UI, interactive maps, bright accent colors, and organized typography.
